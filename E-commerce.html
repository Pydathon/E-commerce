Variables¶

  • InvoiceNo : Numro d'achat, pour chaque nouvelle facturation, un nouveau numro est gnr.
  • StockCode : Numro de transaction, pour chaque objet achet, un numro est affect, la fin de celui-ci, on peut retrouver une lettre donnant des indications sur l'achat, par exemple C signifie que la transaction a t annule.
  • Description : Donne une courte description de l'objet achet.
  • Quantity : Quantite achete.
  • InvoiceDate : Date d'achat.
  • UnitPrice : Prix unitaire.
  • CustomerID : Numro de client, est affect lors de la cration du compte, et ne change pas ensuite.
  • Country : Pays dans lequel l'achat a t effectu.

def tokenization_s(sentences): 
    s_new = []
    for sent in (sentences[:]):
        s_token = word_tokenize(sent)
        if s_token != '':
            s_new.append(s_token)
    return s_new

Le principe de la tokenisation est de sparer une phrase en mots, ce qui va permettre d'amliorer la comprhension, de faciliter le nettoyage et donc de pouvoir appliquer cela des algorithmes de Machine Learning. Par exemple, 'WHITE HANGING HEART T-LIGHT HOLDER' deviendra ['WHITE', 'HANGING', 'HEART', 'T-LIGHT', 'HOLDER']

def maj_stop_words(stop_words, suppw):
    """Met en majuscules la liste des stopwords et rajoute optionnellement des mots  la liste"""
    stop_words_new=[]
    for i in stop_words:
        stop_words_new.append(i.upper())
    if suppw != [] : 
        for i in suppw:
            stop_words_new.append(i.upper())
    return stop_words_new

def remove_stopwords(words, suppw):
    """Retire les Stopwords et la ponctuaction"""
    filtered_sentence=[]
    expression="[A-Za-z]*"
    stop_words = maj_stop_words(list(stopwords.words('english')),suppw)
    i=1
    for i in words:
        for w in i: 
            if w not in stop_words and not w.isdigit() and w not in string.punctuation and re.search(expression,w).group(0)==w : 
                filtered_sentence.append(w) 
            
    return filtered_sentence

Pour continuer le nettoyage, il faut retirer la ponctuation, faire en sorte que tous les mots soient en majuscules ou en minuscule afin que lorsqu'ils seront compars, qu'ils ne soient pas considrs comme deux mots diffrents. Mais il reste tout de mme beaucoup de mots inutiles qu'il faut supprimer. Pour cela, nous utiliserons une liste de 'Stopwords', une liste pr-construite de mots qui sont supprimer.
